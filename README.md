**Motor de Base de Datos:** SQL Server 2019

## Descripción

El presente repositorio contiene la solución al examen técnico solicitado por AB Systems S.A. de C.V como parte del proceso de selección para los puestos disponibles dentro de la empresa.

El proyecto fue desarrollado utilizando Microsoft SQL Server 2019 y contempla:

- Creación de base de datos
- Definición de tablas
- Llaves primarias y foráneas
- Inserción de datos de prueba
- Corrección de inconsistencias
- Consultas analíticas
- Aplicación de reglas de seguridad
- Documentación técnica

---

## Requerimientos Técnicos

- SQL Server 2019
- SQL Server Management Studio (SSMS)

---

## Consideraciones

Los datos utilizados son de prueba y simulan un entorno con posibles inconsistencias.  
Las consultas fueron diseñadas considerando que en un entorno real la base de datos puede contener mayor volumen y variaciones en la información.
